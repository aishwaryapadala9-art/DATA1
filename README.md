# Django User Registration Project

## Project Overview
This is a simple Django web application for user registration and login.

The project stores:
- Username
- Email
- Password
- Date of Birth

using Django models and SQLite database.

---

## Features
- Home Page
- About Page
- User Registration
- Login Page
- Django Admin Panel
- Database Storage

---

## Technologies Used
- Python
- Django
- HTML
- CSS
- SQLite

---

## Project Files

### models.py
Stores user details model.

### views.py
Handles:
- Home page
- About page
- Register page
- Login page

### admin.py
Registers models in Django admin panel.

### apps.py
Application configuration file.

### tests.py
Testing file.

---

## How to Run

### Install Django
```bash
pip install django
```

### Run Migrations
```bash
python manage.py makemigrations
python manage.py migrate
```

### Run Server
```bash
python manage.py runserver
```
